## 天地农耕项目
基于LaraCMS 后台管理系统开发

## LaraCMS
https://www.laracms.cn/

## 说明

## LaraCMS 功能模块：
- 用户管理
- 权限管理
- 角色管理
- 站点信息
- 友情链接
- 栏目导航
- 分类管理
- 文章管理
- 页面管理
- 幻灯管理
- 微信公众号管理
- 第三方登录
- 前端 API
- 文章多图，多附件管理


## 环境需求

* Composer
* PHP >= 7.1.3
* OpenSSL PHP 扩展
* PDO PHP 扩展
* Mbstring PHP 扩展
* Tokenizer PHP 扩展
* XML PHP 扩展
* Ctype PHP 扩展
* JSON PHP 扩展
* Mysql 5.7+
> 中文分词会占用内存，请将 `php.ini` 的 `memory_limit` 参数调整至 256M
